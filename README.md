## Команды для запуска проекта

1. `npm install` (Node v22.11.0, npm v10.9.0)

2. `npm run start` или `npm run build`

---

### Стек технологий проекта:

#### Основные зависимости:

-   Vue 3 Composition Api (script setup)

-   Pinia

-   SCSS

#### Dev зависимости:

-   Vite

-   TypeScript, @types-пакеты
